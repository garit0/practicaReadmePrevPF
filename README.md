# Mi proyecto con GitHub Pages

## Propósito
Este proyecto consiste en la creación y publicación de una página web sencilla como práctica del uso de Git y GitHub Pages.  
El sitio funciona como un mini portafolio personal donde presento algunos proyectos tecnológicos, participación en un hackatón de ciberseguridad y experiencias en eventos como Google y Cisco.
